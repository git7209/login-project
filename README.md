# login-project
web 으로 로그인, 회원가입, 메모장 기능 구현
## 로그인 페이지 설명 
회원가입으로 이어지는 버튼과 로그인 버튼 그리고 두개의 텍스트 박스로 비밀번호와 ID 를 입력하면 되는 구조임
## 회원가입 페이지 설명 
회원가입 페이지에는 ID를 입력하는 박스와 PW를 입력하는 박스가 있는데 여기에 ID는 중복해서 생성할 수 없도록 구현 하였음.
또한 비밀번호를 확인하기위한 박스가 있어서 PW를 잘못 설정하는 것을 방지할 수 있음
## 메모장 페이지 설명
메모장 페이지에는 노트를 추가하는 기능, 휴지통, 영어로 바꿀수 있는 버튼 등 여러가지를 구현해 놓았음
